---
title: "Create Enrollment"
api: "POST /api/enrollments"
description: "Create a new enrollment for an existing student in a specific school year"
---

## Request Body

<ParamField body="student_id" type="uuid" required>
  ID of the existing student to enroll
</ParamField>

<ParamField body="school_year_id" type="uuid" required>
  ID of the school year for this enrollment
</ParamField>

<ParamField body="grade_level" type="string" required>
  Grade level for the enrollment (e.g., "1°", "2°", "10°", "11°")
</ParamField>

<ParamField body="group_code" type="string">
  Group or class code (e.g., "A", "B", "1A")
</ParamField>

<ParamField body="shift" type="string">
  School shift (e.g., "morning", "afternoon", "evening")
</ParamField>

<ParamField body="status" type="string" default="pending_documents">
  Initial enrollment status. Common values:
  - `pending_documents` - Awaiting document submission
  - `active` - Enrollment is active
  - `completed` - Enrollment completed
</ParamField>

<ParamField body="simat_status" type="object">
  SIMAT (Colombian education system) status data
</ParamField>

<ParamField body="extra_data" type="object">
  Additional custom data for the enrollment
</ParamField>

## Response

Returns the created enrollment object with all fields populated.

<ResponseField name="id" type="uuid">
  Unique enrollment identifier
</ResponseField>

<ResponseField name="school_id" type="uuid">
  School identifier
</ResponseField>

<ResponseField name="student_id" type="uuid">
  Student identifier
</ResponseField>

<ResponseField name="school_year_id" type="uuid">
  School year identifier
</ResponseField>

<ResponseField name="grade_level" type="string">
  Grade level
</ResponseField>

<ResponseField name="group_code" type="string" nullable>
  Group or class code
</ResponseField>

<ResponseField name="shift" type="string" nullable>
  School shift
</ResponseField>

<ResponseField name="status" type="string">
  Current enrollment status
</ResponseField>

<ResponseField name="status_history" type="array">
  Array of status changes with timestamp and user who made the change
</ResponseField>

<ResponseField name="simat_status" type="object">
  SIMAT status data
</ResponseField>

<ResponseField name="extra_data" type="object">
  Additional custom data
</ResponseField>

<ResponseField name="student_name" type="string">
  Full name of the student
</ResponseField>

<ResponseField name="school_year_name" type="string">
  Name of the school year
</ResponseField>

<ResponseField name="documents" type="array">
  Array of enrollment documents (empty for new enrollments)
</ResponseField>

<ResponseField name="created_at" type="datetime">
  Timestamp when the enrollment was created
</ResponseField>

<ResponseField name="updated_at" type="datetime">
  Timestamp when the enrollment was last updated
</ResponseField>

## Validation Rules

- The student must exist and belong to the school
- The school year must exist and belong to the school
- A student cannot have multiple enrollments for the same school year
- The status will be automatically tracked in the `status_history` array

## Permissions Required

- `write:enrollment`
- `write:all`
